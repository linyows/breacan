Installation
------------

Add this line to your application's Gemfile:

```ruby
gem 'breacan'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install breacan

Usage
-----

example:

```ruby
require 'breacan'
Breacan.access_token = 'xoxp-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx'
Breacan.channels
=> [{:id=>"C03QXXXX",
    :name=>"hello",
    :is_channel=>true,
    :created=>1424735886,
    :creator=>"U03PXXXX",
    :is_archived=>false,
    :is_general=>false,
    :is_member=>false,
    :members=> ["U0NLXXXX",...
Breacan.channel_by_name('hello').id
=> "C03QXXXX"
```
